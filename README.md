# Fast Food Ordering System

![Java](https://img.shields.io/badge/Java-17-orange)
![MySQL](https://img.shields.io/badge/MySQL-8.0+-blue)
![NetBeans](https://img.shields.io/badge/NetBeans-IDE-green)

A **desktop-based fast food ordering system** built with **Java Swing** and **MySQL** that allows customers to place orders through a kiosk while enabling staff and administrators to manage payments, orders, and menu items efficiently.

---

## Features

- Customer self-service kiosk ordering  
- Dine-in and take-out options  
- Order customization  
- Payment processing  
- Menu management  
- Order logs and reporting  
- Role-based access control  

---

## Technologies Used

- Java Swing  
- MySQL  
- Apache NetBeans  
- Apache Ant  
- JasperReports  

---

## Installation

1. Clone the repository  
```bash id="87j85m"
git clone https://github.com/yourusername/FastFoodOrderingSystem.git
```

2. Import the database from the provided SQL file
   - Open **MySQL Workbench**.
   - Go to the **Administration** tab > **Data Import/Restore**.
   - Select **Import from Self-Contained File** and choose the `KioskDB.sql` file from this project.
   - Set the **Default Target Schema** to `KioskDB`
   - Click **Start Import**.
4. Configure the database connection
5. Open the project in NetBeans
6. Build and run the project
